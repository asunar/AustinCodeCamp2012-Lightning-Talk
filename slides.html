Submitting to App Store

  • Detailed Guide for setting up building iOS apps without a Mac1
  • If you get a 'Can't open config file error' 2
  • Build a Hackintosh in a VirtualBox preferably (10.6.8)3
  • Download and install XCode 4.2 to your Hackintosh
  • Upload your *.ipa file using Application Loader in your Hackintosh
  1. http://community.phonegap.com/nitobi/topics/ detailed_guide_for_setting_up_building_ios_apps_without_a_mac
  2. http://jaspreetchahal.org/warning-cant-open-config-file-usrlocalsslopenssl-cnf/
  3. http://www.sysprobs.com/install-mac-snow-leopard-1063-oracle-virtualbox-32-apple-intel-pc
Apple Notice: Beginning July 22 2010, you will need to use Application Loader to submit your binary to iTunes Connect. Binary uploads will no longer be accepted through the iTunes Connect interface on that date.
  • Cocoa app used to deliver your binary to Apple
  • Offers fast uploading, a more stable connection, and early validation warnings.
  • Runs only on Mac
  • Status of your app in iTunes Connect must be Waiting for Upload before using Application Loader
