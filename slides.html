Goal

Build a cross platform app to solve a REAL problem and submit to App Store / Google Play

  • Scratch your own itch.
  • Diversify skills, bragging rights
  • Time is very scarce when you are a parent. Not enough time to master Java / Objective C
  • Prefer to bet on open technologies like HTML5, JS and CSS etc.

Development Stack

Part I

Tools, Languages & Frameworks that cost NOTHING

  • HTML5 (LocalStorage via Backbone)
  • JavaScript (Backbone.js, jQuery Mobile)
  • PhoneGap and PhoneGap build
  • Vim (Alternatives: Sublime, WebStorm)
  • TestFlightApp
  • VirtualBox

Check out official documentation / mailing lists. Clone sample app and hack away.

Lessons Learned

  • Keep it SIMPLE. Hell is configuring Vim on Windows
  • Firebug is your friend
  • Test in Firefox to avoid same origin error with JQuery Mobile
  • Test app on the targeted devices
  • Test app on the targeted devices (not a typo)
  • Pick an available app name and stick with it
